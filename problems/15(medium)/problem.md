# Problem 15: Three Sum

## Problem Description

Given an array `nums` of n integers, find all unique triplets in the array which gives the sum of zero.

Note:
- The solution set must not contain duplicate triplets.

Example:

Input: `nums = [-1, 0, 1, 2, -1, -4]`
Output: `[[ -1, 0, 1], [-1, -1, 2]]`

## Constraints

- `0 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`
